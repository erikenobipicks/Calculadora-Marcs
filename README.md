# Objectiu Emmarcació — Calculadora

App de càlcul de pressupostos per a emmarcació.

## Desplegament local
```
pip install -r requirements.txt
python app.py
```

## Variables d'entorn
- `SECRET_KEY` — clau secreta Flask (obligatòria en producció)
- `PORT` — port (Railway/Render l'assignen automàticament)
